Mining Sales Management Desktop App
A desktop app built using Electron, React, and SQL to streamline sales entry, cash flow tracking, client management, fleet logs, and expenses for a mining-related business. Designed for real operational use within a Kerala-based small business.

---

Project Overview
The goal was to replace fragmented Excel-based workflows with a consistent, centralized platform that:
- Stores data reliably  
- Reduces manual errors  
- Provides structured sales, payments, and cash flow logs  
- Prepares the business for future AI-assisted analysis  

---

Architecture
Frontend:  
- React + Material UI  
- Custom dark theme  
- Modular pages (Sales, Clients, Cash Flow, Expenses, Fleet, Audit)

Backend:  
- SQLite database  
- SQLAlchemy-style queries  
- Local persistence  

Additional Features: 
- Excel import (Data I/O)  
- Audit logging  
- Timestamps & validation rules  
- Ready for AI-reporting integration  

---

Tools & Technologies
- React  
- Electron  
- Material UI  
- SQLite  
- Node.js  
- Python (for optional data utilities)
